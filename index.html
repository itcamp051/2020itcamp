HTML이란?

HTML은 웹에서 사용하는 문서를 만드는 언어입니다.

CSS란?

문서를 시각화하기 위해 웹 상에서 약속한 언어가 바로 CSS입니다.

자바스크립트란?

웹브라우저에서 실행하는 스크립트 언어입니다. 또한, 웹 브라우저에서 동적인 페이지를 만들기 위해 탄행하였습니다.

제이쿼리란?

자바스크립트의 생산성을 향상시켜 주는 자바스크립트 라이브러리로, 자바스크립트를 좀 더 쉽게 사용하기 위해 만들어 졌습니다.

Ajax란?

Asynchronous JavaScript and XML의 약자로 자바스크립트를 이용해서 비동기식으로 서버와 통신하는 방식을 말합니다.

느낀점 IT 영재캠프 참여 후 느낀점을 이야기해 보았습니다.

  1. Q1디자인 수업 후 느낀점

    김도윤
    이전에 사진을 수정, 보정 등 할 수 없는게 많았는데 캠프를 통하여 무료로 사진을 수정할 수 있는 픽슬러라는 사이트도 알게 되고 픽슬러의 여러기능을 손에 익히게 되어 뿌듯했다.
    내가 나의 개인 페이지를 꾸미는 활동을 하며 다른 친구들의 작품을 보고 느끼며 나의 실력을 더 기를 수 있고 친절하신 선생님께서 설명도 잘 해주셔서 많이 유익했던것 같다.

    김정현
    포토샵을 어느 정도는 알고 있었 지만 제대로 해본 적은 없었기에 이번 디자인 수업에서 포토샵을 사용해본 것이 새로웠다.
    다양한 기능을 쓰며 사진을 편집하는 것이 재밌었고 픽슬러의 기능에 대해 많이 알게 되어서 좋았다.
    사진 편집하는 것을 좋아해서 픽슬러를 앞으로도 많이 활용할 것 같습니다

    윤지원
    디자인 수업을 들으면서 나도 보정을 할 수 있다는게 놀라웠습니다

    이예은
    디자인 수업은 평소 관심있던 분야라 그런지 작업하는 과정이 재미있어서 좋았다.

  2. Q2SW수업 후 느낀점

    김도윤
    평소에 sw에 많은 관심이 있고 알아보고 있었는데 visual studio code를 이용하여 내가 내 손으로 직접 웹페이지를 만든다는것이 신기하고 새롭게 다가왔다.
    이 캠프를 통해 웹 페이지를 만드는 것에 많은 관심을 가지게되었고 기회가 된다면 선생님께서 알려주신 사이트에서 관련된 것을 배우고 싶다는 생각이 들었다.

    김정현
    웹사이트를 제작하는 과정이 궁금했지만, 너무 복잡하고 어려워 보여서 제대로 알려고 한 적이 없었는데 이번에 직접 보게 되니 생각보다 어렵지 않고 너무 재밌었다.
    정말 비주얼 스튜디오에 나와 있는 글자들은 영화에 있을법한 느낌이었고, 또 간혹 잘못 누르던 쓸데없다고만 생각했던 F12키가 이렇게 쓰인다는 걸 처음 알게 되어 내가 뭔가 대단한 사람이 된 것만 같았다.

    윤지원
    SW수업때는 수업이 재미있었는데 실제로 할때는 어려워서 당황스러웠다.

    이예은
    그리고 SW수업은 따라가기 어려웠지만 완성본을 만들고 나니 뿌듯했다.

  3. Q32020 it영재캠프에 참여 후 느낀점

    김도윤
    나와 같은 관심사를 가진 친구들이 많다는 것을 알게되어 기뻤고 내 진로에 관련된 여러가지 경험을 쌓은것같아 알찬 캠프 활동을 했던 것 같다.

    김정현
    정말 시간 가는 줄 모르고 하루종일 작업을 했고 하는동안 너무 재밌었다.
    진로가 어느 쪽인지 아직 확신이 없었는데 이번에 내가 it 쪽 분야에 관심이 많다는 걸 알게 되었고 진로가 어느 정도 굳혀진 것 같다.

    윤지원
    이번 기회로 나도 할 수 있다는 자신감을 가지게 된 것 같아서 매우 뿌듯하고 앞으로는 어려워 보이더라도 한번 시도는 해봐야겠다는 생각이 들었다.

    이예은
    2020 미림 it캠프를 통해 평소 관심이 있었지만 배우지 못했던 분야를 배울 수 있어서 좋았고 it와 미림마이스터고에 한 번 더 관심을 가지게 되는 계기가 되었다.

  4. Q42020 it영재캠프를 하며 가장 기억에 남는 점

    김도윤
    내가 내 손으로 직접 코딩을 하여 내 웹페이지의 일부분을 내가 설정하고 만들었던 것이다.
    앞으로도 이러한 많은 경험을 쌓아 많은 관련 지식을 얻고 싶다.

    김정현
    웹 퍼블리셔, 웹 개발자, 웹 디자이너에 관심이 많아서 참여하게 되었는데 어떤식으로 하는지 알게되어 좋았다.
    특히 쓸데없다고 생각했던 F12가 이렇게 쓰인다는걸 배워서 새로웠다.

    윤지원
    VScode로 코드를 입력했을 때 내가 전문가가 된 것같은 기분이 들어서 가장 기억에 남는다.

    이예은
    vscode에 처음 코드를 입력하고 창을 새로고침 했을 때가 가장 기억에 남는다.
    내가 직접 웹페이지를 만들고 있다는 생각에 뿌듯하고 재미있었다.
